Brand Guidelines

Ledgr

Financial clarity for freelancers. Track invoices, taxes, and cash flow in a dashboard built for the way independent professionals actually work.

01 — Archetype

Brand Personality

The archetype defines Ledgr's emotional tone, voice, and visual direction. Every design choice flows from this foundation.

The Sage + Explorer

Analytical Informed Independent Ambitious

Ledgr embodies the Sage's analytical depth with the Explorer's independent spirit. It provides clear financial intelligence without talking down to its users, respecting that freelancers already understand their work — they just need sharper tools to see the numbers clearly.

The Sage drives 80% of visual decisions: clean grids, precise spacing, restrained palette, and data-rich layouts. The Explorer adds ambition, forward motion, and a refusal to look like yet another banking app.

Voice Guidelines

Tone

Direct and precise. Avoid filler words. Respect the user's time and intelligence.

Language

Use financial terms naturally, without jargon walls. Explain when context demands it, never when it doesn't.

Personality

A sharp colleague, not a corporate advisor. Confident without being condescending. Warm without being casual.

Never

Never use exclamation marks in UI copy. Never say "simple" or "easy" — let the interface prove it.

02 — Logo & Wordmark

Wordmark Variants

The Ledgr wordmark uses Outfit at 800 weight. The missing "e" is the identity element — rendered as a subtle baseline marker in the primary color, evoking a cursor or data entry point. The gap makes the name feel like shorthand, a variable name, something that belongs in a code editor.

06 — Don't

Negative Rules

These are explicit violations of Ledgr's design system. Each pattern undermines the brand's credibility in the financial space.

Don't
Ledgr

Never stretch or distort the wordmark. A warped logo communicates carelessness — fatal for a finance product.

Don't
Ledgr

Never apply gradients to the logo. The wordmark must always be a flat, solid color to work across all contexts.

Don't
Ledgr + Ledgr

Never use the error color for the brand mark. Red/error in finance implies loss, debt, or failure.

Don't
Your quarterly tax estimate is based on projected income for the current fiscal period.

Never set body text below 14px (0.875rem). Financial data demands readability — small text erodes trust.

Don't

Never remove focus indicators from interactive elements. Keyboard navigation is required for accessibility.

Don't
LEDGR

Never use the brand fonts at extreme decorative sizes or in all-caps for display. The wordmark has a specific form — respect it.
