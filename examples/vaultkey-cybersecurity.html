Brand Guidelines v1.0

VAULTKEY

Zero-trust cybersecurity for enterprise. Every perimeter is a liability. Every identity is a surface. Trust nothing. Verify everything.

01 — Archetype

Brand Personality

VAULTKEY operates at the intersection of mastery and disruption. The brand projects the confidence of a Hero archetype with the edge of an Outlaw. It does not ask for trust. It eliminates the need for it.

The Hero + Outlaw

Impenetrable Precise Disruptive Commanding

The Hero archetype drives VAULTKEY's core promise: mastery over chaos, the confidence to protect what matters, the competence that CISOs demand. The Outlaw influence injects an edge that says the old perimeter model is dead and VAULTKEY killed it. This is not a brand that reassures. It confronts. It operates in the gap between what legacy systems promised and what they actually deliver.

Voice Guidelines
Tone
Direct, authoritative, unsentimental. Statements, not suggestions.
Vocabulary
Technical precision over marketing abstraction. Zero-trust, not "enhanced security."
Cadence
Short sentences. Declarative. Occasional fragments for impact.
Never
Never use "we make it easy." Never soften the message. Never promise 100% security.

02 — Logo & Wordmark

Identity System

The VAULTKEY wordmark is a typographic mark set in Cabinet Grotesk Black. "VAULT" carries the weight. "KEY" is set in the primary green to signify the access control layer. All-caps is mandatory. The word break is the identity element.

06 — Don't

Negative Rules

These are explicit violations of the VAULTKEY brand system. Every item here has been rejected for a specific reason. Do not deviate.

Don't
VAULTKEY

Never stretch, skew, or distort the wordmark. Maintain original proportions at all times.

Don't
VAULTKEY

Never apply gradients to the wordmark. The mark must always render as flat, solid color.

Don't
VaultKey

Never use unapproved typefaces. Never render in mixed case. The wordmark is always all-caps Cabinet Grotesk.

Don't
V

Never reduce the brand to a letter inside a rounded square. VAULTKEY is a typographic wordmark. There is no icon.

Don't
Session tokens expire after 3600 seconds. Contact admin for renewal.

Never set body text below 14px (0.875rem). Readability is a security requirement. Unreadable text causes operational errors.

Don't
VAULTKEY

Never use unapproved color combinations. The brand palette exists for a reason. Friendly, warm colors undermine the authority VAULTKEY requires.
